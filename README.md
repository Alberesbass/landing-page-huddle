## Como Usar

1. Clone este repositório:

```bash
git clone https://github.com/Alberesbass/landing-page-huddle.git
```

2. Navegue até o diretório do projeto:

```bash
cd landing-page-huddle
```

3. Abra o arquivo `index.html` no seu navegador da web.
